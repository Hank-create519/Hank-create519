## Current Status (v1.0)

A pure local rule engine `estimateRoundCount()` has been deployed, located in `src/sdk/engine.ts`. It uses three-factor scoring (signal word hits, dimension count, content length) to directly output 1–10 rounds. Zero API calls, millisecond-level response.
